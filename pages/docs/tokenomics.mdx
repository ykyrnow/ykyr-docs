## 3 · Token Utility

$WWW is the utility token of YKYR’s data economy.

### 3.1 Earn & Distribute
- $WWW tokens are distributed to data contributors based on Proof-of-Contribution (PoC).  
  Each uploaded file receives a PoC score weighted by multiple metrics (For detailed scoring methodology, see our [Proof of Contribution](/docs/proof-of-contribution))

### 3.2 Governance
$WWW is DATVotes-enabled – one token equals one vote. Holders can:
- Propose changes (schema, fee %, treasury spend, etc).  
- Vote on-chain; quorum = 2 % of circulating $WWW and a simple majority (> 50 % “For”) passes.  
- Execute after a 48-hour timelock.

### 3.3 Integration with Vana Data Access Architecture
- Buyers can pay with $VANA or $WWW tokens to access data from our DLP
- Uses Vana's standard data refinement and query infrastructure (VRC-15 compliant)
- Integrates with Vana's TEE for secure data processing
